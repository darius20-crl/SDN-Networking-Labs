# SDN Networking Labs

## Overview

This repository contains Software Defined Networking (SDN) projects completed using Mininet, Ryu Controller, OpenFlow 1.3, and network automation concepts.

## Technologies Used

- Mininet
- Ryu Controller
- OpenFlow 1.3
- Linux
- Python
- Wireshark

## Projects

### Mininet Topology Configuration
- Single-switch topologies
- Linear topologies
- Connectivity testing

### Ryu Controller Development
- Learning switch implementation
- OpenFlow rule management
- Network traffic control

### Firewall Implementation
- Host-to-host traffic filtering
- Access control policies
- Flow rule configuration

### Network Testing
- Ping testing
- Throughput analysis
- Link failure simulation

## Skills Demonstrated

- Network Administration
- SDN Concepts
- Linux Networking
- Python Automation
- Troubleshooting
- Network Security
